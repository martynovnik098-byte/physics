
**Задача 1000. A+B Problem**

*   **Суть:** Найти сумму двух целых чисел.
*   **Логика:** Чтение двух переменных и вывод их суммы. Самая простая задача на базовый ввод-вывод.
*   **Сложность по времени:** $O(1)$ — одна арифметическая операция.
*   **Сложность по памяти:** $O(1)$ — храним только две входные переменные и результат.
*   **Алгоритм:** Линейный вычислительный процесс без условий и циклов.
*   **Тип данных:** `int` достаточно для стандартных ограничений большинства систем (до $\pm 2 \cdot 10^9$).

```
#include <iostream>

int sum(int a, int b)
{
	return a + b;
}

int main()
{
	int a, b;
	std::cin >> a >> b;
	int c = sum(a, b);
	std::cout << c;
	return 0;
}
```
