
**Задача 1785. Трудности локализации**

*   **Суть:** Перевести число монстров $N$ в текстовую категорию ("few", "several" и т.д.) по таблице диапазонов.
*   **Логика:** Используется цепочка условий `if — else if`.
    *   Проверка идет **по убыванию** (от 1000 вниз).
    *   Это упрощает код: если сработало `n >= 500`, мы уже точно знаем, что `n < 1000` (иначе сработал бы первый `if`), поэтому писать двойные условия не нужно.
*   **Алгоритм:** Разветвляющийся алгоритм (условный выбор).
*   **Сложность по времени:** $O(1)$ — количество проверок фиксировано (максимум 8 сравнений), не зависит от входного числа.
*   **Сложность по памяти:** $O(1)$ — храним только одно число `n`.

```
#include <iostream>

int main() {
    int n; 
    
    std::cin >> n;
    
    
    if (n >= 1000) {
        std::cout << "legion" << std::endl;
    }
    else if (n >= 500) {
        std::cout << "zounds" << std::endl;
    }
    else if (n >= 250) {
        std::cout << "swarm" << std::endl;
    }
    else if (n >= 100) {
        std::cout << "throng" << std::endl;
    }
    else if (n >= 50) {
        std::cout << "horde" << std::endl;
    }
    else if (n >= 20) {
        std::cout << "lots" << std::endl;
    }
    else if (n >= 10) {
        std::cout << "pack" << std::endl;
    }
    else if (n >= 5) {
        std::cout << "several" << std::endl;
    }
    else {  
        std::cout << "few" << std::endl;
    }
    
    return 0;
}
```
