Вот супер-краткая выжимка для шпоры:

**Задача 1293. Эния**

*   **Суть:** Посчитать площадь $N$ панелей с **двух** сторон.
*   **Формула:** `N * A * B * 2` (умножаем на 2, т.к. обрабатываем обе стороны).
*   **Сложность по времени:** $O(1)$ — нет циклов, всегда выполняется ровно 3 умножения.
*   **Сложность по памяти:** $O(1)$ — используем 4 переменные `int`, объем памяти не растет.
*   **Тип данных:** `int` подходит (макс. результат 2 млн, а `int` вмещает 2 млрд).

```
#include <iostream>
int square(int a, int b, int c)
{
	return a * b * c * 2;
}
int main()
{
	int a, b, c;
	std::cin >> a >> b >> c;
	int d = square(a, b, c);
	std::cout << d;
	return 0;
}
```
