Вот краткая выжимка для шпоры:

**Задача 1409. Два бандита**

*   **Суть:** Найти, сколько банок **не** прострелил каждый из бандитов.
*   **Логика:**
    1.  Гарри прострелил `a`, Ларри `b`.
    2.  Последнюю банку прострелили они оба, значит, она посчитана дважды.
    3.  **Всего банок:** `Total = a + b - 1`.
    4.  Гарри не прострелил: `Total - a` (или `b - 1`).
    5.  Ларри не прострелил: `Total - b` (или `a - 1`).
*   **Сложность по времени:** $O(1)$ — простая арифметика, нет циклов.
*   **Сложность по памяти:** $O(1)$ — используется фиксированное число переменных.

```
#include <iostream>

int main()
{
	int a, b;
	std::cin >> a >> b;
	int a_1 = a+b-1-a;
	int b_1 = a+b-1-b;
	std::cout << a_1 << ' ' << b_1;
	return 0;
}
```
